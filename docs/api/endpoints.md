# API Endpoints

## Authentication
- POST /api/auth/login
- POST /api/auth/logout

## Projects
- GET /api/projects
- GET /api/projects/:id
- POST /api/projects
- PUT /api/projects/:id
- DELETE /api/projects/:id

## Profile
- GET /api/profile
- PUT /api/profile
- POST /api/profile/avatar
- POST /api/profile/resume

## Messages
- GET /api/messages
- POST /api/messages
- PUT /api/messages/:id/read
- DELETE /api/messages/:id
